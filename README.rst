==============
Whippersnapper
==============

An emoticon and shorthand message translator. 

Installation
============

``pip install -e git+https://github.com/jondbaker/whippersnapper.git#egg=whippernsapper``

Requirements
------------
* Python >= 2.7

Usage
=====
``$ ./translate.py "459! Can't wait to CU :-)."``

Tests
=====
``$ ./tests/unit.py``
